There are two disjoint sets called "Host" and "Link".

Each element of "Link" is related by "from" to exactly one element of "Host", and by "to" to exactly one element of "Host".

For every element in Link, Link also contains the reverse of the "link".  The reverse of a link is an element whose "from" value matches that "to" value of the link and whose "to" value matches the "from" value of the "link".
