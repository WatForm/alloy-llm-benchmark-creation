There are two disjoint sets called "Host" and "Link".

Each element of "Link" is related by "from" to exactly one element of "Host", and by "to" to exactly one element of "Host".

The reverse of a link is a "Link" whose "from" value matches that "to" value of the link and whose "to" value matches the "from" value of the "link".
For every element in "Link", the set "Link" also contains the reverse of the "link".
