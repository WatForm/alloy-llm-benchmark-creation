There is a set of elements called "N".

Each element of "N" is related by "suivant" to a set of elements of "N". 

The relation "suivant" is acyclic over "N". 

There do not exist elements of "N" such that one is related to the other by both a direct "suivant" link and are also related to the same other element by a path of length at least two using "suivant".
