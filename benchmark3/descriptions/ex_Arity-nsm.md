There are three sets named A, B, and C.

Each element of A is associated by r with a relation from B to C. For any element of A, the r relation contains only pairs whose first element is in B and whose second element is in C.

For every element of A, r is a partial function from B to C: for each element of B, there is at most one element of C that it is related to by r. A given element of B may be related to no element of C, and if it is related to one, it is related to exactly one.
