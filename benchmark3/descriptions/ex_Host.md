There are two sets, "Host" and "Link".

Each element of "Link" is related by "from" to exactly one element of "Host", and by "to" to exactly one element of "Host".

For every element of "Link", there exists some element of "Link" whose "from" value is equal to the first link’s "to" value and whose "to" value is equal to the first link’s "from" value. This means every link has at least one reversed link in "Link".
