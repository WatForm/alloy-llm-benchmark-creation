There are sets named "Card" and "Suit".

Each element of "Card" is related by "suit" to exactly one element of "Suit".

There is a set named "hand" whose elements are all elements of "Card".

The condition named "ThreeOfAKind" requires that "hand" contains exactly 3 elements, and that the set of all "suit" values of elements in "hand" contains exactly 1 element.
